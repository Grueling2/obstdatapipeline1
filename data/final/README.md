This folder is the target location for Jupyter Notebook file gold_daily_eod_report.ipynb reports. Each simulated business day is partitioned into a separate final Excel file which contains the five metrics:
1. Total sales today
2. Sales over time, split up by store
3. Top book sold today
4. Biggest single increase, and biggest single decrease in popularity of a book over the preceding 7 days.
5. Top 5 best-selling authors over the preceding 7 days.
